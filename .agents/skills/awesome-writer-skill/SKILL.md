# GetBiji 写作技能 (Writer Skill)

多风格智能写作技能，支持专业、科普、极简三种写作风格，专门负责内容生成和创作。

## 核心功能

- **多风格写作**：支持专业、科普、极简三种写作风格
- **智能风格识别**：根据用户需求自动选择合适写作风格
- **内容融合**：整合研究结果生成高质量内容
- **引用管理**：自动处理引用和来源标注
- **格式控制**：保持输出格式的一致性
- **质量检查**：确保内容的准确性和可读性

## 写作风格详解

### 1. 专业风格

**适用场景**：学术论文、技术文档、研究报告、医学综述

**语言特征**：
- 严谨用词，避免口语化表达
- 大量使用专业术语和缩写
- 逻辑结构清晰，层次分明
- 数据支撑，引用规范
- 客观陈述，避免主观判断

**格式要求**：
- 标题层级清晰（H1-H4）
- 段落结构严谨
- 表格图表规范
- 参考文献格式

**示例**：
```markdown
## 方法学
本研究采用随机对照试验设计，纳入2023年1月至2023年12月期间确诊的胰腺癌患者共156例。所有患者均签署知情同意书，研究方案经伦理委员会批准（批件号：XXX-2023-001）。

### 统计学分析
采用SPSS 26.0软件进行统计分析。计量资料以均数±标准差(x̄±s)表示，组间比较采用独立样本t检验；计数资料以[n(%)]表示，组间比较采用χ²检验。P<0.05为差异有统计学意义。
```

### 2. 科普风格

**适用场景**：公众号文章、博客分享、内部培训、患者教育

**语言特征**：
- 通俗易懂，避免过度专业化
- 适当使用比喻和类比
- 案例驱动，故事性强
- 互动性强，引导思考
- 保持趣味性

**格式要求**：
- 标题吸引人
- 段落短小精悍
- 适当使用emoji
- 信息图解辅助

**示例**：
```markdown
## 胰腺癌治疗新进展：就像给癌细胞上了一把"智能锁"

最近，科学家们开发出了一种新型靶向药物，就像是给狡猾的癌细胞上了一把"智能锁"。这把锁非常聪明，只会对特定的癌细胞起作用，而不会伤害正常的细胞。

🎯 **简单来说**：这种药物能精准识别癌细胞表面的特殊标记，就像钥匙开锁一样精准。
```

### 3. 极简风格

**适用场景**：社交媒体、摘要速览、要点总结、快问快答

**语言特征**：
- 短句为主，直击要点
- 关键词突出
- emoji点缀
- ≤500字原则
- 信息密度高

**格式要求**：
- 简洁标题
- bullet points
- 重点突出
- 便于分享

**示例**：
```
🔬 胰腺癌新药进展
• 靶向治疗突破：精准打击癌细胞
• 副作用降低：患者生活质量提升  
• 生存期延长：中位OS达15.2个月
• 检测创新：血液ctDNA监测
#胰腺癌 #靶向治疗 #新药研发
```

## 风格路由规则

```
用户指令 → 风格识别 → 风格选择 → 内容生成

识别关键词:
├─ "论文/报告/文档/学术" → 专业风格
├─ "公众号/博客/分享/科普/教育" → 科普风格  
├─ "小红书/摘要/速览/要点/总结" → 极简风格
└─ 默认 → 科普风格
```

## 内容生成流程

### 1. 输入解析
- **内容源识别**：确定要写作的主题和素材来源
- **风格要求解析**：识别用户的风格偏好
- **受众分析**：根据内容类型确定目标受众

### 2. 结构规划
- **大纲生成**：基于内容源生成写作大纲
- **段落分配**：合理分配内容比例
- **重点突出**：确定核心要点

### 3. 内容撰写
- **风格适配**：按选定风格进行内容撰写
- **引用整合**：合理引用研究结果
- **质量控制**：确保内容准确性

### 4. 格式输出
- **格式化处理**：按要求格式输出
- **质量检查**：最终质量审核
- **文件保存**：保存到指定目录

- `GET_KNOWLEDGE_BASES`: 笔记知识库配置

## 错误处理

| 问题类型 | 处理方式 |
|----------|----------|
| 内容源不足 | 提示用户补充信息或调整写作范围 |
