# 写作风格指南

## 风格对比速查

| 维度 | 专业风格 | 科普风格 | 极简风格 |
|------|----------|----------|----------|
| 人称 | 第三人称 | 第二人称 | 第一人称 |
| 语气 | 客观严谨 | 亲切对话 | 活泼直接 |
| 段落 | 长段落 | 短段落 | 列表项 |
| emoji | ❌ 禁用 | ✅ 适度 | ✅ 大量 |
| 引用 | 必须完整 | 简化标注 | 仅关键点 |
| 长度 | 2000-5000字 | 1500-3000字 | ≤500字 |

### 专业风格
```
✅ RMC6236联合RTK抑制剂的临床研究进展与应用前景
✅ 基于SCAN框架的多源证据整合研究方法
❌ 震惊！这个药居然...
```

### 科普风格
```
✅ RMC6236联合用药：理论很美好，现实怎么样？
✅ 限流方案避坑指南：我踩过的3个大坑
❌ 论RMC6236的临床应用（太学术）
```

### 极简风格
```
✅ 🔥 6236联合用药｜3个关键点
✅ 💊 胰腺癌靶向药速查
❌ 关于RMC6236联合用药的一些思考（太长）
```

## 结论写法

```

```

### 科普风格
```
总结一下：联合用药方向是对的，但现在数据还不够多。我的建议是：
- 保持关注，但不要盲目
- 有条件可以咨询医生参与临床试验
- 先把现有方案用好
```

### 极简风格
```
💡 结论：有希望，但要等更多数据
📝 建议：咨询医生，持续关注
```

## 禁忌事项

### 专业风格
- ❌ 使用"震惊"、"竟然"等情绪词
- ❌ 省略引用来源
- ❌ 给出没有证据支持的结论

### 科普风格
- ❌ 堆砌专业术语
- ❌ 段落超过5行
- ❌ 没有个人案例或故事

### 极简风格
- ❌ 超过500字
- ❌ 长段落描述
- ❌ 缺少emoji和分隔线
